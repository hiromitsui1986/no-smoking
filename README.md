### 使い方
トップページ「START」ボタンから入り銘柄、値段を入力してください。再び「START」をクリックするとタイマーが起動します。
  
### 開発環境
ruby/rubyonrails/JQuery/MySQL/github/Vicual Studio Code/heroku
